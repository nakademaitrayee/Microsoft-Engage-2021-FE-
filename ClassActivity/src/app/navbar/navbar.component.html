CLASS ACTIVITY








Welcome To The Class Activity App!



This is an interactive app for users who want to create and play different quizzes/surveys with their friends,family or colleagues. The user once signed in, can see the Navigation panel on the left. It can be used to log out of this application.

The user can then select options of creating a survey or playing a survey. The 'Take survey' button takes the user to his/her own set of created surveys. Inside that, the user can see 'Take all Surveys' button, which in turn shows the user, the surveys, created by other users. Then on clicking on any survey, the user can cast his/her vote. After succesful submission, they can see how they have performed against all the other users who have taken the same survey.

So what are you waiting for? Go create some cool new surveys and share it with your friends! We hope you will enjoy the app!
